# README


* Ruby version 3.1.2
* Rails version 7

* System dependencies
  
    postgresql 13+

* Database creation
    rails db:setup

* Deployment instructions in localhost:
  
    1- check you are running ruby 3.1.2
  
    2- git clone this repo
  
    3- cd game_of_life
  
    4- bundle install
  
    5- rails db:setup
  
    6- rails s
